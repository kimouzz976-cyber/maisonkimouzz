Le projet Maison Kimouzz

Un écosystème agro-économique à Mayotte visant à développer l’autonomie alimentaire, la production locale et une économie circulaire durable.

🌍 Vision globale

Maison Kimouzz n’est pas seulement une boutique. C’est une structure complète qui connecte agriculture, pêche, élevage, logistique et distribution locale.

🚀 Objectifs

- Réduire les importations alimentaires
- Valoriser les producteurs locaux
- Créer des emplois à Mayotte
- Digitaliser la vente alimentaire locale

📦 Système futur (base de données + commande)

Dans la prochaine étape :

✔ Produits stockés dans une base de données (JSON / Firebase)
✔ Ajout/modification en temps réel
✔ Commandes enregistrées automatiquement
✔ Tableau admin connecté
✔ Paiement en ligne

🧭 Évolution du projet
